# ✂️ Companhia da Beleza - Sistema de Agendamento Online

Este é um projeto de uma Landing Page e fluxo de agendamento desenvolvido para o salão **Companhia da Beleza**. O design foi inteiramente baseado em um protótipo do **Figma**, focando em uma experiência de usuário (UX) elegante e moderna com *Dark Mode*.

---

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5**: Estruturação semântica do site.
* **CSS3**: Estilização personalizada, uso de Flexbox, Grid e variáveis (Design Tokens).
* **JavaScript**: Lógica de navegação entre etapas de agendamento e integração com a API do WhatsApp.
* **Figma**: Ferramenta utilizada para o design de interface (UI).
* **Linux Mint**: Sistema operacional utilizado no ambiente de desenvolvimento.

---

## 🛠️ Funcionalidades

- [x] **Landing Page**: Apresentação visual do salão com foco em conversão.
- [x] **Fluxo de Agendamento**: Escolha de serviços, profissionais, data e horário.
- [x] **Integração com WhatsApp**: Envio automático dos dados do agendamento para o profissional.
- [ ] **Painel Administrativo**: (Em breve) Para gestão de horários.
